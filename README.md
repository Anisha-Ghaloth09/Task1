# Responsive Landing Page

## Project Overview

This project is a **Simple Responsive Landing Page** built using **HTML5** and **CSS3**. It demonstrates the fundamentals of modern web design, including page structure, responsive layouts, Flexbox, and media queries.

The landing page contains a navigation header, a hero section with a call-to-action button, and a footer with social media links. The layout adapts to different screen sizes, making it suitable for desktops, tablets, and mobile devices.

---

## Features

* Responsive design for different screen sizes
* Clean and modern user interface
* Header with logo and navigation menu
* Hero section with heading, description, and call-to-action button
* Footer with social media links
* CSS Flexbox layout
* Media queries for mobile responsiveness
* Hover effects for navigation links and buttons

## Getting Started

### Prerequisites

* Visual Studio Code
* Google Chrome (or any modern browser)
* Live Server Extension for VS Code

### Installation

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Ensure the following files are present:

   * `index.html`
   * `style.css`
4. Install the **Live Server** extension if it is not already installed.
5. Right-click on `index.html` and select **Open with Live Server**.

## Responsive Design

The website uses CSS media queries to provide a better user experience on smaller devices.

* Desktop: Horizontal navigation menu
* Tablet: Adjusted typography and spacing
* Mobile: Navigation links stacked vertically
